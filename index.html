Tester igjen hvordan lokale sider kan bli sendt over til GitHub ... Hva må jeg gjøre for at det skal skje?

Slik jeg har gjort det til nå:

  • Opprettet en konto på GitHub
  • Lastet ned GitHub Desktop
  • Lastet ned Atom og aktivert GitHub der også
  • Testet meg frem med varierende resultat

Så langt ...

  • Når man gjør endringer i en fil (kan være hvilken som helst fil) så vil det komme beskjed om 'Unstaged Changes' i Git-panelet
  • Når man vil 'lagre', kan man si 'Stage All' og det blir gjort klart for lagring
  • Du må skrive en kort beskjed om hva du har gjort
  • Så kan du gå til neste steg, som er 'Commit to main'
  • Helt til sist så velger du 'push' for å legge det opp på Git-server
